## üìù Documentation Standards

### File Naming Conventions:
- **UPPERCASE-WITH-DASHES.md** - Major documentation files
- **lowercase-with-dashes.md** - Supporting documentation
- **Phase-X-NAME.md** - Phase implementation guides

### Location Guidelines:
- **`docs/`** - Active, user-facing documentation
- **`data/docs/`** - Development documentation and planning
- **`docs/agents/`** - AI agent persona files
- **`docs/archive/`** - Historical/superseded documentation
- **`docs/features/`** - Feature-specific documentation

### When to Archive:
- Documentation is superseded by a newer version
- Feature has been deprecated or removed
- Information is no longer accurate but valuable for history
- After 30+ days of being marked as outdated
